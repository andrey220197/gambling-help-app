# Telegram Mini App: Помощь при проблемном гемблинге

## Цель
Безопасное, анонимное и научно обоснованное приложение для помощи людям с проблемными финансовыми импульсами, игрой/ставками, сильной тягой и нарушением контроля.

**Важно:** Приложение НЕ диагностирует и НЕ лечит. Оно помогает отслеживать состояние, снижает импульсивность, даёт научные объяснения и помогает при кризисных состояниях.

## Ключевые функции (MVP)

### 1. Онбординг (3-4 экрана)
- Что за приложение?
- Как оно помогает?
- Как обеспечена анонимность?

### 2. Ежедневный чек-ин
- Тяга (0-10)
- Стресс (0-10)
- Настроение (0-10)
- Был ли срыв (да/нет)
- Заметка (опционально)

### 3. Трекер "Дни без игры"
- Текущий streak
- Рекорд
- История

### 4. SOS-план ("Мне тяжело сейчас")
- Дыхательная техника (box breathing)
- Пауза 10 минут
- Grounding 5-4-3-2-1
- Переключение внимания

### 5. Научные карточки (10-15 штук)
Категории:
- Дофамин и мотивационная система
- Неопределённость и почти-победа
- Когнитивные ловушки
- Триггеры и сенситизация
- Эмоциональное выгорание

### 6. Профиль
- Дата регистрации
- Количество чек-инов
- Recovery code (формат: WOLF-MOON-7234)
- Настройки

## Telegram Bot
- `/start` — приветствие + кнопка "Открыть приложение"
- `/app` — ссылка на Mini App
- `/help` — информация + службы помощи
- Напоминания о чек-ине
- SOS-уведомления при высоком риске

## Анонимность (критично!)
```
anon_hash = HMAC-SHA256(telegram_id + SECRET_SALT)
```
- telegram_id НЕ сохраняется
- Пользователь идентифицируется только по hash
- Recovery code для восстановления доступа

## Юридические ограничения
**НЕ использовать термины:**
- зависимость
- лудомания  
- лечение
- диагностика

**Везде добавлять дисклеймер:**
> Это не медицинская помощь. Если у вас кризис — позвоните 112.

## Связь с Telegram-каналом "Нейроблок"
- Карточки в приложении могут ссылаться на развёрнутые посты
- Кросс-промо между каналом и приложением
- Единый научный стиль подачи информации
